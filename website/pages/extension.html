Core Features

How the Extension Works

👁️

Real-time Detection

MutationObserver watches the DOM and processes newly injected comments instantly — zero polling delay.

⚡

Batch Processing

Comments are batched into groups of 20 and sent every 1.5s — 200 comments → 4 API calls instead of 200.

💾

Client-side Cache

Previously scored comments are never re-sent to the API. Identical comments across sessions get instant results.

🎚️

Sensitivity Slider

Users control the toxicity threshold via the popup UI (0–100). Preferences saved via chrome.storage.sync.

👆

Click-to-Reveal

Blurred toxic comments can be revealed on-demand by clicking. Warning badge shows toxicity score.

🔒

Privacy First

Comments are sent only to your local backend (127.0.0.1:8000). No data leaves your machine.

Installation

Load the Extension

1

Clone the Repository

git clone https://github.com/AnandRawat11/
Toxic-comments-detector.git
2

Open Chrome Extensions

Navigate to chrome://extensions/ and enable Developer mode.

3

Load Unpacked

Click Load unpacked and select the extension/ folder.

4

Start the Backend

cd backend
uvicorn main:app --reload --port 8000
5

Browse & Shield

Visit YouTube, Reddit, or Instagram. Toxic comments are automatically blurred with a warning badge.
